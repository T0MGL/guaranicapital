# Directorio Public

Este directorio contiene archivos estáticos que estarán disponibles públicamente.

## Brochure PDF

Coloca tu brochure en PDF aquí con el nombre:
- **guarani-brochure.pdf**

Para subir el archivo, simplemente copia o mueve tu PDF a este directorio:

```bash
cp /ruta/a/tu/brochure.pdf public/guarani-brochure.pdf
```

## Hosting

Cuando subas el proyecto a tu hosting (Netlify, Vercel, etc.), este directorio `public/` se copiará automáticamente a la raíz del sitio web.
